# Agent instructions

## Supported agents

This project uses Codex and agent-neutral workflows. Do not create or recommend Claude-specific configuration, including `CLAUDE.md`, unless the user explicitly changes this decision.

## Agent skills

### Issue tracker

Issues and PRDs are tracked in GitHub Issues. External PRs are not a triage request surface. See `docs/agents/issue-tracker.md`.

### Triage labels

The project uses the canonical triage labels documented in `docs/agents/triage-labels.md`.

### Domain docs

This is a single-context repository. Read the root `CONTEXT.md` and relevant ADRs under `docs/adr/` when they exist. See `docs/agents/domain.md`.
